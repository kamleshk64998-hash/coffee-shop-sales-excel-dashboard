# ☕ Coffee Shop Sales Dashboard using Microsoft Excel

## 📌 Project Overview

This project focuses on analyzing Coffee Shop Sales data and creating an interactive dashboard using Microsoft Excel. The objective of this project is to transform raw sales data into meaningful insights that help understand customer purchasing behavior, product performance, and business trends.

## 🎯 Objectives

* Analyze sales performance and customer trends.
* Identify top-selling products and categories.
* Compare sales across different locations.
* Determine peak business hours.
* Evaluate product size preferences.
* Create an interactive dashboard for decision-making.

## 🛠 Tools and Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning and Transformation
* Dashboard Design

## 📊 Key Analysis Performed

### 1. Sales Report Dashboard

Created an interactive dashboard to provide a complete overview of sales performance and key business metrics.

### 2. Product-wise Quantity Analysis

Analyzed the quantity sold for each product to identify best-selling and low-performing products.

### 3. Location-wise Sales Analysis

Compared sales across different store locations to identify high-performing branches.

### 4. Hour-wise Sales Analysis

Examined customer purchasing patterns across different hours to determine peak sales periods.

### 5. Size-wise Sales Analysis

Analyzed customer preferences based on product sizes and their contribution to overall sales.

### 6. Product Category Analysis

Evaluated the performance of different product categories and their impact on total sales.

## 📈 Key Insights

* Identified peak sales hours and customer traffic patterns.
* Determined top-performing products and categories.
* Compared revenue contributions from different locations.
* Created an interactive dashboard for business decision-making.
